## What Audio Cue Mapper Does

Audio Cue Mapper is a browser-first analysis utility built to help with cue organization in motion design and sync-oriented projects. It emphasizes fast visual checking, cue positioning, and timeline reading so you can convert media into structured timing data without bringing in a separate desktop application stack.

The tool is aimed at workflows where audio analysis needs to feed directly into animation software. Because processing stays client-side and export includes After Effects JSX, it suits both individuals and teams that want a lighter way to inspect waveforms, identify events, and move into production more efficiently.

- Browser-based audio and video file analysis

- Peak picking and tiering for organizing cue candidates
